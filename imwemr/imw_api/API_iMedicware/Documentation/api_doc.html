1. Overview

1.1. Authorization

You will need these parameters :
USER_KEY : User Key to connect (provided by imwemr);
SECRET_KEY : Secret Key phrase to connect (provided by imwemr);
USER_AGENT : Static value (iMedicWareApiclient) ;

You need to post 3 header variables for every API request.
User-Agent, X-Key parameters and sec_code for secure Authorization.
sec_code is md5 value

Please see the example for Authorization code preparation.
Example Authorization code for PHP : 

<?php
$data_to_sign = self::USER_KEY.self::USER_AGENT.self::SECRET_KEY;
$secure_code = md5($data_to_sign);

$headers = array();
$headers[] = "User-Agent: " . self::USER_AGENT;
$headers[] = 'X-Key: '.self::USER_KEY;
$headers[] = 'sec_code: '.$secure_code;

//using headers in  CURL OPTIONS
curl_setopt($curl_session, CURLOPT_HTTPHEADER, $headers);

?>
