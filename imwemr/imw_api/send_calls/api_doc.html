1.4. Authorization

You will need these parameters :
USER_KEY : User Key to connect (provided by MDprospects);
SECRET_KEY : Secret Key phrase to connect (provided by MDprospects);
USER_AGENT : Static value (MDprospectsAPIclient) ;

You need to post 2 header variables for every API request.
User-Agent and X-Api-Signature parameters for secure Authorization.

Please see the example for Authorization code preparation.
Example Authorization code for PHP : 

<?php

$time_stamp=date('YmdHis');
$data_to_sign = USER_KEY.USER_AGENT.$time_stamp.SECRET_KEY;

$signature = base64_encode(sha1($data_to_sign, true));

$headers = array();
$headers[] = "User-Agent: " . USER_AGENT;
$headers[] = 'X-Api-Signature: '.USER_KEY."|$time_stamp|$signature";

//using headers in  CURL OPTIONS
curl_setopt($curl_session, CURLOPT_HTTPHEADER, $headers);

?>
